Partition String ⭐⭐

Given a string S of lowercase letters, partition S into as many as parts so that one letter only appear in one part. Return the partitions as a list.

Example 1:

Input:

S = “bbeadcxede”

Output:

[“bb”, “eadcxede”]

Example 2:

Input:

S = “baddacx”

Output:

[“b”, “adda", “c”, “x”]
